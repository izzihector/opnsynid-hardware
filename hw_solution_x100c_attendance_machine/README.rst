=============================================
HW Proxy - Solution X100-C Attendance Machine
=============================================

This module allows user to connect posbox with Solution X100-c Attendance
machine

=====
Usage
=====
Launch the Odoo server with:`./odoo.py --load=...,hw_solution_x100c_attendance_machine`

Installation
============

No Installation Required

Bug Tracker
===========

Bugs are tracked on `GitHub Issues
<https://github.com/open-synergy/opnsynid-hardware/issues>`_.
